## Theme

- The theme will be announced at the start of the jam
- Your game should incorporate the theme in some way
- Creative interpretations are encouraged!

---

## Submission Requirements

Your submission must include:

1. **A playable build** of your game
   - Windows build preferred
   - Web builds (HTML5) are also accepted
   - Mac/Linux builds are optional extras

2. **Source code** via your GitHub repository
   - Make sure your repository is public or judges have access

3. **A short description** of your game
   - What is the game about?
   - How does it relate to the theme?
   - How to play (controls)

4. **Credits** for your team
   - Team name
   - Member names and roles

---

## What You CAN Use

- This template (Unity/Godot/GameMaker projects)
- Game engine built-in features and assets
- Your own pre-made code/assets 

---

## What You CANNOT Do

- Submit a game started before the jam
- Use assets downloaded from the internet or created before the jam
- Submit someone else's game as your own
- Continue working after the deadline

---

## Code of Conduct

- Be respectful to other participants
- Help others if you can
- Have fun!
