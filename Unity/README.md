# Unity Project

This is a minimal Unity project template for the Dundee & Angus Game Jam.

## How to Open

1. Open **Unity Hub**
2. Click **Add** > **Add project from disk**
3. Navigate to this `Unity` folder and select it
4. Unity Hub may prompt you to install the matching Unity version or upgrade the project
5. Click on the project to open it

## First Time Setup

When Unity opens this project for the first time, it will:
- Generate the `Library` folder (this is gitignored)
- Create default project settings
- Import the default packages

This may take a few minutes on first open.
